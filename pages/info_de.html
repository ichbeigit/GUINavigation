Der verwendete Name für eine Navigation muss einmalig sein! Das gilt nicht nur innerhalb eines Navigationstyps.


In den Code wird eine Navigation so eingesetzt: 

<?php GUINavigation::factory('meine_navigation'); ?>

Navigationstypen

  • Statisch
    Generiert eine Navigation aus der vorhandenen Struktur nach den eingegeben Parametern.
  • Contextuell
    Generiert eine Navigation aus der vorhandenen Struktur abhänig vom aktuell angezeigten Artikel und den eingegebenen Parametern.
  • Einfach
    Erlaubt beliebige Artikel auf einer Ebene als Navigation auszugeben.
  • Sprachnavigation
  • Brosamennavigation

Der unter "System" ausgewählte Startartikel wird immer übersprungen, egal wo er sich befindet. Der Startartikel kann über die Option "Website-Startarikel einbinden" am Anfang oder am Ende auf der obersten Ebene der Navigation eingebunden werden.

Konfigurations Optionen

  • Kategorien oder Artikel auflisten?
    Die Möglichkeit Artikel in der Navigation auszugeben ist interessant für einfach aber sehr breite Navigationen. Bei tiefen, komplizierten Navigationen kann es zu überraschenden Ergebnissen kommen, im besonderen im Zusammenspiel mit einer Brosamennavigation.
    Bei der Listung von Artikeln werden Kategorien durch ihre Kategorien-Startartikel auf der Ebene gelistet auf der diese sind und nicht auf der Ebene auf der der Kategorieeintrag ist.
  • Auf erste Unterkategorie verlinken
    Wird eine Kategorie bzw. der Startartikel einer Kategorie ausgewählt, dann wird bei der Ausgabe des Links der Name der ausgewählten Katregorie ausgegeben, aber mit der ersten Unterkategorie verlinkt, sofern es eine gibt. Der Menüpunkt wird mit der Klasse "link-first-subcat" markiert (siehe auch "Ausgabe").
    Funktioniert nicht für einfache Artikel.
