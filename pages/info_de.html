Der verwendete Name für eine Navigation muss einmalig sein! Das gilt nicht nur innerhalb eines Navigationstyps.


In den Code wird eine Navigation so eingesetzt: 

<?php GUINavigation::factory('meine_navigation'); ?>

Navigationstypen

  • Einfach
    Erlaubt beliebige Artikel auf einer Ebene als Navigation auszugeben.
  • Statisch
    Generiert eine Navigation aus der vorhandenen Struktur nach den eingegeben Parametern.
  • Contextuell
    Generiert eine Navigation aus der vorhandenen Struktur abhänig vom aktuell angezeigten Artikel und den eingegebenen Parametern.
  • Sprachumschalter
  • Brosamennavigation

Der unter "System" ausgewählte Startartikel wird immer übersprungen, egal wo er sich befindet. Der Startartikel kann über die Option "Website-Startarikel einbinden" am Anfang oder am Ende der Navigation eingebunden werden.

Konfigurations Optionen

  • Kategorien oder Artikel auflisten?
    Die Möglichkeit Artikel in der Navigation auszugeben ist interessant für einfach aber sehr breite Navigationen. Bei tiefen, komplizierten Navigationen kann es zu überraschenden Ergebnissen kommen, im besonderen im Zusammenspiel mit einer Brosamennavigation.
    Bei der Listung von Artikeln werden die Kategorien-Startartikel auf der Ebene gelistet auf der sie sind.
